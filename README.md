## 📥 How to Run

1. **Download the files**:
   - Clone this repository:
     ```bash
     git clone https://github.com/Giacomo-Marconi/2048Game
     cd 2048-classic
     ```
   - Or download the ZIP and extract it.
2. **Open the game**:
   - Double-click on `index.html` to open it in your browser.
3. **Start playing**!
